# CLI Usage Guide

This document provides guidance on using the collation-statemachine CLI.

## Installation

```bash
pip install -r requirements.txt
```

## Basic Usage

### Running the State Machine

```bash
python run_agent.py --input <input_file> --output <output_file>
```

### Running Tests

```bash
pytest tests/
```
